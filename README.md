
# Todo Task Card

A responsive and accessible Todo Task Card built with plain HTML, CSS, and JavaScript, focused on clean UI, semantic structure, and testability.

## Live Demo

🔗 [https://increaze.github.io/testable-todo-card/](https://increaze.github.io/testable-todo-card/)

---

## Features

* Task details (title, description, priority, status)
* Dynamic time remaining (updates every 30s)
* Completion toggle with UI state update
* Tags with `data-testid` support
* Accessible buttons (Edit, Delete)
* Fully responsive (320px–1200px)

---

## Tech Stack

* HTML5
* CSS
* JavaScript
* GitHub Pages

---

## Run Locally

```bash
git clone https://github.com/Increaze/testable-todo-card.git
cd testable-todo-card
```

Open `index.html` in your browser.
